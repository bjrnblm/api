PageService

PageService provides access to the /pages endpoint of the NPO API

Parameters:
Name Type Attributes Default Description
basePath String <optional>
 Taken from the API server config as 'npoApiServer' The API server basepath up until the media endpoint or just the protocol and host part when taken from the config, no trailing slash ( e.g. https://apihost.nl ).
apiKey String <optional>
 Taken from the API server config as 'npoApiKey' Your NPO API key
secret String <optional>
 Taken from the API server config as 'npoApiSecret' Your NPO API secret
