Class: FormatDuration

FormatDuration

Provides numerous ways to return a duration's value

Constructor

new FormatDuration(duration, separatoropt)

Parameters:
Name Type Attributes Default Description
duration Number The duration in milliseconds
separator String <optional>
 ':' The separator used in the string returning methods of FormatDate

Methods

getHoursMinutesSeconds() → {String}

Returns:
- A formatted string of hours:minutes:seconds, where : is either the default or through the constructor passed in separator, and where minutes and seconds are leading zeroed up to two digits.
Type
String
Example
var formatDuration = new FormatDuration( 4205000 );
 formatDuration.getHoursMinutesSeconds() // returns 1:01:05

getMilliseconds() → {Number}

Returns:
- The original duration in milliseconds
Type
Number

getMinutes() → {Number}

Returns:
- The number of minutes in this duration
Type
Number
Example
var formatDuration = new FormatDuration( 124000 );
 formatDuration.getMinutes() // returns 2

getMinutesSeconds() → {String}

Returns:
- A formatted string of minutes:seconds, where : is either the default or through the constructor passed in separator, and where seconds are leading zeroed up to two digits.
Type
String
Example
var formatDuration = new FormatDuration( 124000 );
 formatDuration.getMinutesSeconds() // returns 2:04

 var formatDurationWithSeparator = new FormatDuration( 124000, '\'' );
 formatDurationWithSeparator.getMinutesSeconds() // returns 2'04
